Multiple Distances Plugin

Multiple Distances Plugin is an extension of Gama-q2 for excluding outlying distances based on Grubb's two-side test.

Quick Start

Click on push button and exclude some value in column test. If you have only one or two values you cannot make Grubb's two-sided test. If you are testing slope distances it loads only measurements with zero height of signals (observations or targets).
